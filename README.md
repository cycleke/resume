# 编译

首先需要安装 `stack` 和 `texlive-full`（也推荐使用 `TinyTeX`），
之后使用脚本 `update.sh` 来更新简历

```bash
# Generate the pdf files
./update.sh

# Generate the pdf files and push to GitHub
./update.sh -g
```
